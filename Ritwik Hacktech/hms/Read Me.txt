Description of Project

We have made a web app name Hospital Management System. This web app is made with HTML, CSS, JAVASCRIPT, PHP and MySQL  Database.
This website helps people to find hospitals nearby to their homes and how many beds they have to admit the patients.
We have also a Covid-19 dashboard which fetch real-time data from covid-19 api. It tells you how many cases are recorded till Database.
Our website conatins three buttons one for patient login/signup, one for doctor login/signup and last one for admin.
As soon as patient select the preferred hospital he/she can book an appointment to see doctor.  We are currently working on project Our
project is approximately 90 percent completed

How to run the Hospital Management System (HMS) Project
1. Download the  zip file

2. Extract the file and copy hospital folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name hms

6. Import hms.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/hospital (frontend)
